Контейнеры:

	Main Container
Контейнер, состоящий из компоненты HEADER и 2х блоков с выбором мужского либо женского гардероба, в середине каждого блока находится компонент Button, по нажатию, которой осуществляется переход на Garderob Container.

	Garderob COntainer
Контейнер, состоящий из компоненты HEADER и компоненты GoodsNavigation, а так же блока в котором находится компонент Slider

	Goods Container
Контейнер, состоящий из компоненты HEADER, компоненты GoodsNavigation, компоненты GoodstInfo
Блок-полосы, в которой указано колличество моделей товара в общем и сортировка по цене или размеру.
под блоком-полосой массив одинаковых компонент StuffCard.
	
	Stuff Container
Контейнер, состоящий из компоненты HEADER, компоненты GoodsNavigation,
Блока в котором находится компонент Slider, cправа от slider находится компонент StuffInfo.
Cлева от slider находится ссылка НАЗАД, по нажатию осуществляется перход на Goods Container
и страница скролится до того товара, на котором был осуществлён переход.


	Cart Container
Контейнер, состоящий из компоненты HEADER, компоненты GoodsNavigation,
Блока в котором находится заголовок "Моя корзина", под заголовком находится
массив с компонентами CartCard. Справа от блока находится маленький блок
с С заголовком "итого : сумма" и компоненты Button	
	
Компоненты:

	HEADER
Компонент хеадер состоит из компонента логотип, поиска и двух навигационных ссылок
Ссылок на корзину и на личный кабинет пользователя.

	GoodsNavigation
После того как пользователь выбрал для себя гардероб (мужской или женский)
появляется навигация, где находится 3 навигационных ссылки Аксессуары, Одежда, Обувь.
При наведении на компонент GoodsNavigation, появляется компонент GoodsNavigationActive (изначально он скрыт).

	GoodsNavigationActive
При навидении на одну из нав.ссылок задний фон затемняется и появляется
компонент GoodsNavigationActive. В состав входит 4 одинаковых компонента - GoodsDirrectory.
По нажатию на один из этих компонент осуществляется переход на 

	StuffCard
Компонент, в котором есть панель с ценой и компонента Button, всё это обёрнуто блоком, фоном
которого является фотография товара.
Так же по нажатию на Button осуществляется переход на Stuff Container. 
	
	StuffInfo
Компонент, состоящий из Загловка, цены, размера, который можно выбрать. Так же компонента Button


	GoodsDirrectory
Компонент, состоящий из кнопки на фоне которой находится блок с фотографией.

	GoodstInfo
Компонент, состоящий из заголовка и текста, обёрнутые в блок с серым фоном.
